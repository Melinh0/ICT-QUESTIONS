ENUNCIADO:  
9. Which of the following statements about Nginx is false?

ITENS:  
A) Nginx features low memory usage, high concurrent connections, and fast response.  
B) Nginx implements functions such as HTTP server, virtual host, and reverse proxy.  
C) The configuration of Nginx is simple.  
D) Nginx is good at handling both dynamic and static requests.

ITEM CORRETO: D

EXPLICAÇÃO:  
Option A: This is a true statement. Nginx is a high-performance HTTP and reverse proxy server. It is known for its high efficiency, lightweight, and high configurability. It can handle a large number of concurrent connections and uses very low memory usage.  
Option B: This is a true statement. Nginx can be configured as an HTTP and HTTPS server and reverse proxy server, and it also supports virtual hosting. It allows hosting websites on the basis of multiple websites or applications.  
Option C: This is a true statement. Although the Nginx configuration file is not as simple as some other web servers, it is still relatively simple and flexible. If you are familiar with the syntax and instructions of Nginx, it provides a large number of documents and community support to help you configure it.  
Option D: This is a false statement. Nginx is mainly used to handle static requests. It is not as efficient as Apache or other web servers when it comes to handling dynamic requests. Apache is better suited for handling dynamic requests, such as those generated by server-side scripts or frameworks. Nginx mainly features static request handling capabilities and reverse proxy functions. Apache is more suitable for dynamic request handling.
